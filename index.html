ResponseStar

ResponseStar is a full-stack survey analytics application which uses MeteorJs, React and MongoDB. The website accepts surveys from LimeSurvey and SurveyMonkey and allows useers to tag freetext answers. Finally, the system shows an analysis of question answers as well as how questions were tagged. This system is privately owned and therefore the source code is unavailable.

Ct-Wrap

Ct-wrap is a serverless wrapper site used to make the information on https://www.clinicaltrials.gov/ easier to read. It uses NodeJS and React to connect to their api in the background and display the trial information in a more user friendly format. It also features the ability to download trial data in various formats.

Audiomata

Audiomata is a tool made for the Unity game engine. It allows audio to be managed in more flexible manner by allowing users to tag audio clips and then perform a search through those tags. It also has features to control most audio components using a command and factory pattern. E.g. You can make a reverb effect fade in gradually.
